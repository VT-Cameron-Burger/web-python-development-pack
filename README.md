# Web & Python Development Pack

A curated extension pack for VS Code that includes essential extensions for HTML/CSS/JavaScript and Python development.

## What's Included

This extension pack includes the following extensions:

### Python Development
- **Python** (`ms-python.python`) - Core Python language support
- **Pylance** (`ms-python.vscode-pylance`) - Fast, feature-rich Python language server
- **Python Debugger** (`ms-python.debugpy`) - Python debugging support
- **Black Formatter** (`ms-python.black-formatter`) - Python code formatting with Black
- **isort** (`ms-python.isort`) - Python import organization

### Web Development (HTML/CSS/JS)
- **Prettier** (`esbenp.prettier-vscode`) - Code formatter for HTML, CSS, JS, and more
- **ESLint** (`dbaeumer.vscode-eslint`) - JavaScript and TypeScript linting
- **HTML CSS Support** (`ecmel.vscode-html-css`) - CSS IntelliSense for HTML
- **Auto Rename Tag** (`formulahendry.auto-rename-tag`) - Automatically rename paired HTML/XML tags
- **Auto Close Tag** (`formulahendry.auto-close-tag`) - Automatically add HTML/XML close tags

### CSS Tools
- **Stylelint** (`stylelint.vscode-stylelint`) - CSS/SCSS/Less linting

### AI Assistance
- **GitHub Copilot** (`github.copilot`) - AI pair programmer
- **GitHub Copilot Chat** (`github.copilot-chat`) - AI chat features

## Installation

1. Install this extension pack from the VS Code marketplace
2. All included extensions will be automatically installed
3. Restart VS Code if prompted

## Usage

Once installed, all the extensions will be available and configured for optimal web and Python development:

- Python files will have full IntelliSense, debugging, and formatting support
- HTML/CSS/JS files will have autocompletion, linting, and formatting
- Use Prettier to format your code (Ctrl+Shift+P â†’ "Format Document")
- Use ESLint to catch JavaScript errors and style issues
- Take advantage of GitHub Copilot for AI-assisted coding

## Requirements

- VS Code 1.105.0 or later
- For Python development: Python 3.6 or later installed on your system
- For Copilot features: GitHub Copilot subscription
